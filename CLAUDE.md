# Project Instructions

## Project

Personal portfolio website.

## Goals

* Premium and modern UI.
* Clean code and maintainable architecture.
* Excellent performance.
* Mobile-first responsive design.
* Accessibility best practices.

## Tech Stack

* Next.js 16
* React 19
* TypeScript
* Tailwind CSS v4
* Framer Motion

## Code Style

* Prefer reusable components.
* Avoid code duplication.
* Keep files small and focused.
* Use TypeScript strict typing.
* Preserve existing visual design unless explicitly requested.

## Design Principles

* Inspired by Linear, Raycast, Vercel and modern SaaS websites.
* Minimalistic.
* High attention to spacing and typography.
* Subtle animations only.
* Dark theme first.

## Workflow

Before making changes:

1. Explain what will be modified.
2. List affected files.
3. Wait for confirmation if the change is large.

After making changes:

1. Explain what changed.
2. Explain why.
3. Mention any possible side effects.
